# Workshop: DevSecOps com SDLC

Bem-vindo ao laboratório prático de DevSecOps! 🚀

Este repositório foi criado para demonstrar a importância da segurança automatizada no Ciclo de Vida de Desenvolvimento de Software (SDLC).

## 🎯 Objetivo do Lab
Identificar e corrigir vulnerabilidades em dependências de código (SCA - Software Composition Analysis) utilizando o Snyk.

## 🛠 Pré-requisitos
- Conta no [GitHub](https://github.com/)
- Conta no [Snyk.io](https://snyk.io/)

## 📝 Passo a Passo
1. **Fork:** Faça um fork deste repositório para sua conta pessoal.
2. **Conexão:** Conecte o repositório ao seu painel no [Snyk.io](https://snyk.io/).
3. **Scan:** Aguarde a análise automática detectar as vulnerabilidades.
4. **Fix:** Utilize a funcionalidade de "Fix this vulnerability" para gerar um Pull Request automático.
5. **Merge:** Revise e aceite o PR para corrigir a segurança do projeto.

## ⚠️ Aviso
Este repositório contém uma dependência propositalmente vulnerável para fins educacionais. **Não utilize este código em produção!**
